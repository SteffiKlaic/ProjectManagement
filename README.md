# ProjectManagement

ASP.NET Core MVC web application for managing projects and tasks.

## Technologies
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- ASP.NET Identity
- System.Net.Http.Json (for API communication)

## Features
- User registration and login
- Authentication with ASP.NET Identity
- User-specific projects
- Project-specific tasks
- CRUD operations for projects and tasks
- Project and task status system
- Soft delete functionality
- Project and task filtering
- AI-powered project and task description generation via OpenRouter API
